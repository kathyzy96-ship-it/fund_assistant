# Fund Invest Assistant Skill

一个用于生成基金次日正式日报的 Codex Skill。它读取本地基金持仓表，在次日 09:00 使用前一交易日正式净值生成日报，优先展示当日涨跌，再跟踪累计盈亏、预警和资讯变化。

## 适合做什么
- 每个交易日次日早上复盘基金表现。
- 区分“当日涨跌”和“累计浮盈亏”，避免把累计亏损误判成单日亏损。
- 跟踪组合中接近预警线或已经触发预警的基金。
- 结合行业新闻、基金经理观点和资产方向，给出谨慎的观察建议。

## 数据口径
- 正式日报只使用前一交易日正式单位净值。
- 盘中估值只用于临时看盘，不进入次日正式日报主账。
- QDII 等净值滞后的基金必须单独标注净值日期。
- 报告顺序固定为：当日涨跌、累计盈亏与预警、资讯提炼、策略建议。

## 文件说明
- `SKILL.md`：日报生成规则和执行流程。
- `portfolio.example.csv`：持仓表模板，不包含真实持仓。
- `portfolio.csv`：本地真实持仓文件，建议不要提交到公开仓库。

## 持仓表格式
请在本地创建 `portfolio.csv`，字段与 `portfolio.example.csv` 保持一致：

```csv
基金代码,基金名称,买入均价,当前持有份额,预警跌幅
000000,示例基金A,1.0000,1000.00,-5%
```

## 隐私提醒
真实的 `portfolio.csv` 可能包含个人持仓、成本和仓位信息，不建议上传到公开 GitHub 仓库。仓库中只保留 `portfolio.example.csv` 作为模板即可。
